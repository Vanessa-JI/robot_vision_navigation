COMP0130 Robotics Vision and Navigation 
Coursework 1:Integrated Navigation for a Robotic Lawnmower
Authors: Joseph Rowell*, Vanessa Igodifo*, Yuansheng (Jason) Zhang*
*These authors have equal contributions.

This project tracks an autonomous lawnmower thorugh GNSS Kalman Filter
and Dead Reckoning analysis.
See Brief 'COMP0130 Coursework 1 instructions 2022.pdf' for project brief.
See Report 'COMP0130_RVN_Coursework_1_Report.pdf' file for results.

USAGE:
To run, run the cw1_script_2.m file, figures will be generated
and saved to /Results folder.
